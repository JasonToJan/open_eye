# Flutter版本open_eye（Flutter version 2.8.0）

Fluter版本开眼APP：<br>
1、整体项目架构基于Getx搭建，完成路由，依赖注入；<br>
2、网络请求框架基于Retrofit+Dio实现，配合官方JsonSerialize实现解析；<br>
3、封装项目页面多状态（加载中，成功，失败，空数据）；<br>
4、封装列表页面下拉刷新+上拉加载更多等，具体更多内容，请下载项目了解更多<br><br>
